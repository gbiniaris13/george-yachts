# Directories + NAP Pack (Φ2.4)

Αρχή: ΠΟΙΟΤΗΤΑ over ποσότητα. Μόνο δωρεάν, μόνο σχετικά. Τα spammy directories ΒΛΑΠΤΟΥΝ. Το NAP πρέπει να είναι ΠΑΝΟΜΟΙΟΤΥΠΟ παντού (βυτε-προς-byte) — αυτό διαβάζει η Google για consistency.

## Το κανονικό NAP block (copy-paste, ποτέ παραλλαγές)

Name:    George Yachts Brokerage House LLC
Address: Charilaou Trikoupi 190A, Kifisia 145 64, Attica, Greece
Phone:   +30 697 038 0999

**Δεύτερο κύμα (αν θέλουμε κι άλλα, χαμηλότερη προτεραιότητα):** CharterWorld + YachtCharterFleet έχουν paid listings — ΟΧΙ (κανόνας: τίποτα πληρωμένο). Ελληνικά επιμελητήρια (ΕΒΕΑ e-directory, free για μέλη) — αν είναι ήδη μέλος.

## Εκτέλεση

- Τα περισσότερα θέλουν δημιουργία λογαριασμού → χέρι George (απαγορεύεται σε μένα). Με το NAP block από πάνω είναι 5'/directory.
- ΠΡΟΣΟΧΗ στο ωράριο: παντού «Open 24 hours» πλέον — ό,τι παλιό listing λέει 9-6 ή 9-9 διορθώνεται.
- Μετά από κάθε καταχώρηση: μου λες και την επιβεβαιώνω + προσθέτω το profile URL στο sameAs του Organization schema (ενισχύει το entity graph).
